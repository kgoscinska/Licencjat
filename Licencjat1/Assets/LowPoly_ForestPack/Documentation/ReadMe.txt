RC Studio - Low Poly Forest Pack
--------------------------------
Version: 1.0
Publisher: RC Studio
Contact: rcstudiosocial@gmail.com

1. Overview
-----------
This package contains a collection of stylized low poly forest assets for Unity developers.
It is optimized for mobile and PC projects and includes seasonal variations.

2. Contents
-----------
- 20+ Trees (Summer, Autumn, Winter variations)
- Bushes and grass variations
- Rocks (small and big)
- Tree stumps and dead tree
- Demo scene (Forest_Demo.unity)

3. Installation
---------------
1. Import the package into your Unity project.
2. Go to: Assets/LowPoly_ForestPack/Scenes/
3. Open "Demo.unity" to view all assets in action.

4. Usage
--------
- Drag and drop prefabs from Assets/LowPoly_ForestPack/Prefabs into your scene.
- All prefabs are zeroed (Position 0,0,0 / Rotation 0,0,0 / Scale 1,1,1).
- Materials and textures are included in the Materials and Textures folders.
